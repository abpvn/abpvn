## ABPVN Tools with python (3.8+) language

### Tool list:
* Domain change detect: [__domain_change.py](__domain_change.py)
* Domain with no ip address: [__no_ip_domain.py](__no_ip_domain.py)
* File move or deleted: [__file_not_found.py](__file_not_found.py)
* Outdate element hiding rule: [__outdate_element_hide.py](__outdate_element_hide.py)
* Outdate network rule: [__outdate_network_rule.py](__outdate_network_rule.py)

## Setup
* Install python 3.8+
* Create venv with `python -m venv venv`
* Active venv with `source venv/bin/acgtive` on Linux/Unix/GitBash or `venv\Scripts\active` on Windows
* Install dependency with `pip install -r requirements.txt`
* Run tool with `python [tool_name].py`
